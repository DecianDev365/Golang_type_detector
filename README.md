# ⚡ Go Type Detector

A terminal-based type detector built with Go and Charm's Bubble Tea framework.

## Features
- Detects `int`, `float64`, `bool`, and `string` types
- Beautiful TUI powered by Bubble Tea + Lip Gloss
- Two modes:
  - **Endless Mode** — keep detecting until you type `q`
  - **Set Number Mode** — choose how many inputs to test
- Live results shown as you type
- Full results summary at the end

## Built With
- [Go](https://go.dev/)
- [Bubble Tea](https://github.com/charmbracelet/bubbletea)
- [Lip Gloss](https://github.com/charmbracelet/lipgloss)
- [Bubbles](https://github.com/charmbracelet/bubbles)

## Run It
```bash
git clone https://github.com/DecianDev365/Golang_type_detector.git
cd Golang_type_detector
go mod tidy
go run typedector.go
```
